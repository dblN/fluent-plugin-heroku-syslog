# fluent-plugin-in-heroku-syslog

## Component

### HerokuSyslogInput

Plugin to accept syslog input from [heroku syslog drains](https://devcenter.heroku.com/articles/logging#syslog-drains).

## Configuration

```
<source>
  type heroku_syslog
  bind 0.0.0.0
</source>
```
